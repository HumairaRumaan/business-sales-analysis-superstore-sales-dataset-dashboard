# business-sales-analysis-superstore-sales-dataset-dashboard

📊 FUTURE_DS_01 – Business Sales Performance Analytics- Superstore Dataset Dashboard using Power BI

📌 Project Overview
This project was completed as part of the Future Interns Data Science & Analytics Internship Task.

The goal of this project is to analyze business sales data using the Superstore dataset and transform raw data into meaningful insights through an interactive Power BI dashboard.

The dashboard helps understand sales performance, revenue trends, product performance, and regional contribution to support business decision-making.

🎯 Business Questions Addressed
• Which product categories generate the highest revenue?
• How do sales and profits change over time?
• Which regions contribute the most to total sales?
• Which customer segments generate the highest revenue?
• What are the top-performing products?

🧹 Data Preparation
• Cleaned and organized the Superstore dataset
• Removed inconsistencies and ensured correct data formats
• Created calculated fields and KPIs in Power BI
• Built an interactive dashboard for analysis

📊 Dashboard Features
• KPI Cards (Total Sales, Quantity, Profit, Profit Margin)
• Sales by Category Analysis
• Sales by Sub-Category
• Monthly Sales and Profit Trends
• Sales by Customer Segment
• Sales by Payment Mode
• Sales by Shipping Mode
• Interactive Region Filter

🔍 Key Insights
• Office Supplies category generated the highest sales.
• Phones are the top-selling sub-category.
• Consumer segment contributes the largest share of sales.
• COD is the most used payment method.
• Sales show an increasing trend across the months.

🛠 Tools Used

💡 Skills Demonstrated

🚀 Conclusion
This project demonstrates how business sales data can be analyzed using Power BI to identify key performance insights and support data-driven business decisions.
